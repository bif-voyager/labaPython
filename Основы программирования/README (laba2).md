
# Лабораторная работа № 2

## Задание

1. Рассмотреть символьные последовательности длины 6 из пятибуквенного алфавита  
    {K, A, T, E, P}.  
    Найти, сколько таких последовательностей начинаются с буквы **P** и заканчиваются буквой **K**.
	
2. Вычислить выражение
	
    216^6 + 216^4 + 36^6 - 6^{14} - 24
    
    Записать результат в системе счёччисления с основанием 6  
    и определить, сколько различных цифр содержит эта запись.
    
3. Среди натуральных чисел, не превышающих (10^9),  
    найти все числа, соответствующие маске  
    `12345??8?`  
    (символ `?` обозначает одну произвольную цифру),  
    которые делятся на 23 без остатка.  
    Для найденных чисел вывести также результаты деления на 23.

## Результаты вычислений

### Задача 1. Подсчёт количества последовательностей

**Код программы:**

```python
from itertools import product

letters = "KATEP"
count = 0
for middle in product(letters, repeat=4):
    count += 1

print("Количество последовательностей:", count)
```

**Результат выполнения:**

```
Количество последовательностей: 625
```

Так как первая буква фиксирована как `P`, последняя — `K`,  
остаётся 4 свободных позиции, каждая из которых может принимать одну из 5 букв.

N = 5^4 = 625

### Задача 2. Перевод числа в 6-ричную систему

**Код программы:**

```python
def to_base(n, base):
    digits = []
    while n > 0:
        digits.append(n % base)
        n //= base
    return digits[::-1]

value = 216**6 + 216**4 + 36**6 - 6**14 - 24
digits_base6 = to_base(value, 6)
unique = set(digits_base6)

print("Количество различных цифр:", len(unique))
```

**Результат выполнения:**

```
Количество различных цифр: 4
```

Число в шестёричной системе имеет вид:

```
555501555555555520
```

Набор цифр: {0, 1, 2, 5}.  
Следовательно, различных цифр — **4**.

### Задача 3. Маска числа и делимость на 23

```python
results = []

for a in range(10):
    for b in range(10):
        for c in range(10):
            num = int(f"12345{a}{b}8{c}")
            if num <= 10**9 and num % 23 == 0:
                results.append((num, num // 23))

print("Число\tДеление на 23")
for n, d in results:
    print(n, "\t", d)
```
